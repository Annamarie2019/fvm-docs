## Run locally

1. Download, build the site, and start a local server:

    ```shell
    git clone https://github.com/filecoin-project/fvm-docs
    cd fvm-docs
    npm install
    npm run start
    ```

1. Open [localhost:1313](http://localhost:1313/)
